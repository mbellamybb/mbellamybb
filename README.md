### Sales Volume Variance Analysis Dashboard  
<a href="https://public.tableau.com/views/Superstore-SalesVolumeVariance/Superstore-SalesVolumeAnalysis">
      <img src="https://public.tableau.com/thumb/views/Superstore-SalesVolumeVariance/Superstore-SalesVolumeAnalysis" width="25%">
</a>

**Problem**  
Organizations in high-volume industries struggle to detect meaningful sales variance early enough to act. Static reports do not support anomaly identification at scale.

**Approach & Architecture**  
Built a variance monitoring framework using Tableau with configurable anomaly detection logic.

Key architectural elements:
* Parameter-driven variance thresholds
* Box-and-whisker statistical distribution modeling
* Heatmap-based temporal pattern detection
* Interactive filtering across manufacturer, category, and time

The system was designed as a scalable pattern for operational monitoring, not a single-use analysis.

**System Capabilities**  
* Real-time threshold adjustment
* Automated visual flagging of outliers
* Multi-dimensional segmentation
* Drill-down workflows for root cause analysis

**Strategic Value**   
Models how BI tools can function as monitoring systems rather than static reporting interfaces.
